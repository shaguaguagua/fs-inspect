**fs-inspect** is a modern CLI for inspecting and operating multi-instance FreeSWITCH clusters.

The official `fs_cli` connects to one FS at a time. `fs-inspect` sits above a fleet and answers cluster-level questions in a single command:

- *Where is extension 8001 registered right now?*
- *Show me every active call across all nodes.*
- *Which node is leaking channels?*

This site hosts the **build log** — a post every time something interesting ships or breaks — and eventually the command reference.
